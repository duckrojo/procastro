Welcome to timeseries’s documentation!¶

This is the documentation page for the timeseries module for the photometrical analysis of astronomical timeseries. The timeseries package is composed by the following files:

  • timeseries/astrocalc.py
  • timeseries/astrointerface.py
  • timeseries/astroplot.py
  • timeseries/img_scale.py
  • timeseries/__init__.py
  • timeseries/timeseries.py

Requirements¶

  • Dependencies from mathematical modules: numpy, scipy
  • Dependencies from astronomical modules: astropy, dataproc, pyfits
  • Dependencies from graphical modules: ImageTk, matplotlib, PIL, Tkinter
  • Dependencies from modules of the standard library: copy, math, os
